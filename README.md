# Vuln Database

Vulnerability database with CVE lookup.

Database kerentanan dengan pencarian CVE.


## Description

Provides offline CVE lookup and vulnerability tracking for security operations.

## Deskripsi

Menyediakan pencarian CVE offline dan pelacakan kerentanan untuk operasi keamanan.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Search: node vulndb.js search CVE-2024-1234
- Check product: node vulndb.js check nginx 1.18.0
- Demo: node vulndb.js --demo

## Penggunaan

- Cari: node vulndb.js search CVE-2024-1234
- Periksa produk: node vulndb.js check nginx 1.18.0
- Demo: node vulndb.js --demo
